UI/UX Design



Day 1: Introduction to UI/UX Basics

  • UI (User Interface) focuses on the visual elements of a product
    such as layout, colors, typography, and interactive components.
  • UX (User Experience) emphasizes the overall experience of the user
    while interacting with a product, ensuring usability, accessibility, and satisfaction.

Day 2: Figma designing


On the second day, we explored the design tool Figma and its basic functionalities. Key learnings included:
  • Creating and managing frames
  • Designing layouts within frames
  • Adding and editing text elements
  • Applying font size, colors, and alignment
  • Understanding basic design properties and tools

Day 3 & Day 4: Red Bull Project (Design & Prototyping)


During the third and fourth days, we started working on a project based on the Red Bull brand. Activities included:
  • Designing UI screens inspired by the brand theme
  • Creating visually consistent layouts
  • Applying animations to enhance user interaction
  • Building prototypes to simulate real user flow
